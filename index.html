MinePi

High Performance Minecraft Server OS for the Raspberry Pi
Targeted for the Raspberry Pi 3.

Features

- Low energy operation
- Customizable
- Based on Raspbian Lite, apt-get is available.
- Standard Linux Utilities
- Optional Web Server Capability

System Requirements

- Raspberry Pi 1 Model B or higher
- 8GB SD Card or larger
- 2A Power Supply (Samsung charger cables have no stability to the Raspberry Pi)
- An ethernet connection (for high speed internet access)
- A computer with SSH
NOTE: Do not connect it to a monitor, you can do everything from SSH.
